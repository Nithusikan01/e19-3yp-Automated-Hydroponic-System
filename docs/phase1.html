System

The device on the farmer's side is capable of monitoring the sensors and actuators continuously even incase of a power failure. The administrator will be able to reister the users, monitor systems from all the farmers using our product through the Web application in the administrators's side.

High-level System Overview

Method used to ensure secure data transfering:

MQTT protects the user from “Man-in-the-Middle” attacks where hackers can use vulnerabilities in public networks to steal data transmitted to the user. Using TLS encryption to mask a user's connection with the website can prevent this sort of attack.

Flow Control

The system consists of three main endpoints

  • Web application in Administrators's end
  • Mobile application in Farmers's end
  • Hardware in Farmer's end

Hardware Design


The IoT device on the Farmer's side has 2 main components.

  1. A device with Sensors, and Actuators
  2. A microcontroller with UPS - uninterruptible power source to connect with AWS
