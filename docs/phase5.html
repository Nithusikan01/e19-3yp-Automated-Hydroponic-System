Software Testing

Functional requirements is tested under three main stages

  • Unit testing
  • Integration testing
  • End-to-end testing


Hardware Testing

The functionality of the sensors will be tested before connecting them to the circuit.

  • Test the Esp32 for proper functioning with all pins are working.
  • Test the sensors and calibrate them.
  • Test the Pumps and power supplies.

Hardware unit testing and Integration testing

API testing

  • Tools: Postman

Integration testing

  • Tools: Selenium

End-to-end testing

  • Tools: netsparker and NoGotofail
